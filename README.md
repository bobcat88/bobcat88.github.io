# Johan Proust Portfolio

Personal portfolio and CV website for Johan Proust.

## Deployment

This repository is published with GitHub Pages from the root of the `main` branch.

The public website entry point is [`index.html`](./index.html). Keeping the HTML in `index.html` lets GitHub Pages serve it as a normal full-width static page instead of rendering the project README through a Markdown/Jekyll content wrapper.

## Local Preview

Open `index.html` directly in a browser, or serve the repository root with any static file server.

## Assets

- `johan-proust.webp` - profile portrait used by the page.
- `portfolio.png` - portfolio preview image.
- `project_note.md` - implementation notes and patch history.
